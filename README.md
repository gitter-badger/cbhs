# CoolBasic Haskell compiler

[![Join the chat at https://gitter.im/milankinen/cbhs](https://badges.gitter.im/milankinen/cbhs.svg)](https://gitter.im/milankinen/cbhs?utm_source=badge&utm_medium=badge&utm_campaign=pr-badge&utm_content=badge)

This project tries to compile CoolBasic (http://www.coolbasic.com) code with Haskell.


## Setup


Install cabal, then

    cabal sandbox init
    cabal install
    ./scripts/generate


## Run

    cabal run
